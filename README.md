# Susam
#### Susam is an experimental Game Engine which powered by WebGL and Typescript 

URL : https://ek-susam.herokuapp.com/


#### Install & Run Demo
```bash     
    yarn
    yarn bundle
    yarn start
```
* Demo address for local => http://localhost:3333
